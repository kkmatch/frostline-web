# Vape E‑commerce (MVP)

This repo contains a custom vape e‑commerce MVP with:
- `apps/web`: Next.js storefront
- `apps/api`: NestJS backend API

## Setup (placeholder)
Install dependencies at the repo root and run dev servers.

## Environments
- Web: `apps/web/.env.example`
- API: `apps/api/.env.example`
